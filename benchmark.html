Traditional CPU architectures look like they use fewer resources because they run on lower single-thread wattages. But because they process data in a single-file line, they lock up the system over long execution tracks, wasting accumulated energy. By telling the parallel graphics fabric to hop over empty tracks, our framework uses the GPU's native speed while drawing less power than the GPU baseline.
