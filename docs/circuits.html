Circuits, Holds, Routes, Wires

The building blocks of a Sanguis organism

Circuit (Organ)

A circuit is an organ. It holds state, fires on triggers, emits signals, and wires to other circuits. The collection of all circuits and their wiring IS the organism.

circuit Name {
    hold name : layer shape = default    // what this organ carries
    when trigger {                        // when this organ fires
        but condition                     // gate (optional)
        emit value to target              // what comes out
    }
}

In V3, body is an alias for circuit:

body Hello {          // same as: circuit Hello {
  hold greeting : text = "hello"
  when start {
    emit greeting to screen
  }
}

Hold (Neuron Group)

hold name : layer shape = default

Examples:

hold vision     : live    field [448] = 0.0     // current visual input
hold weights    : learned field [448] = 0.5    // learned visual patterns
hold ep001      : memory  field [448]           // a saved snapshot
hold reward     : memory  scalar      = 0.0     // how good that moment was
hold tag        : memory  text         = ""      // what was happening
hold self       : aware   field [448]             // who I am right now
hold confidence : aware   scalar       = 0.0     // how sure I am

A hold is not a variable. It is a neuron or group of neurons. It does not get "assigned." It gets fed by the route.

When (Route Trigger)

when trigger_name {
    // route body — what happens when this circuit fires
}

A when block defines a trigger. The circuit fires when:

1. The route reaches Stage 1 (Sense) and the trigger condition is met, OR
2. Another circuit's emit wires into this when

Every circuit has exactly one route pass per breath. The when determines whether the circuit actively fires during that pass or stays passive (just holding its state, ticking silently).

But (Gate Condition)

when trigger {
    but condition          // only fire if this is true
    but other_condition    // AND this too
    emit value to target    // what comes out
}

but is the gate. It is the C4 layer's voice in the route. A route with no but always fires. A route with but only fires when the condition holds.

but is not an if statement. It is a modulation. In hardware terms, it is a gate on the clock signal. In biological terms, it is attention — the organism decides whether this circuit matters right now.

Conditions:

but x > 0.5                     // numeric comparison
but x < 10.0                    // less than
but x = 0.0                     // equality
but x is not empty              // has content
but x is empty                  // has no content
but self aligned with memory    // C4 ↔ E3 alignment (dot product > threshold)
but disk has e3                 // special: disk has .e3 files
but true                        // always true (explicit)

Emit (Output)

emit value to target

emit is the only way data leaves a circuit. It fires during Stage 7 (Act).

emit state to brain.s1              // feed into another circuit's hold
emit signal to muscle.when_move     // trigger another circuit
emit write(data) to disk "path"     // write to persistent storage
emit print(text) to screen          // output to terminal
emit 42 to world                    // set exit code (native only)
emit "text" to screen               // print a string literal

Arithmetic inside emits:

emit a + b to sum                   // addition
emit a * b to product               // multiplication
emit a * w + b * (1 - w) to blended // weighted blend
emit sum(field) to total            // sum all neurons
emit max(field) to peak             // highest activation
emit dot(a, b) to alignment         // dot product (similarity)
emit norm(field) to magnitude       // vector length

Wire (Connectome)

wire CircuitA.emit_name to CircuitB.when_name

Wiring is the connectome. The collection of all wires between all circuits IS the organism's neural wiring diagram. Changing a wire changes the organism.

wire Vision.emit to Brain.when_sense        // direct wiring
wire Brain.emit_act to Muscle.when_move     // chain
wire Brain.emit_remember to E3Store.when_save  // memory wiring
