Language Reference

Grammar, keywords, layers, shapes, and comments

Grammar (plain English)

genome     := (circuit | wire)*
circuit    := "circuit" IDENT "{" (hold | when | wire)* "}"
hold       := "hold" IDENT ":" layer shape ("=" value)?
layer      := "live" | "learned" | "memory" | "aware"
shape      := "scalar" | "text" | "field" "[" number "]" ("[" number "]")*
when       := "when" IDENT "{" ("but" condition)* ("emit" expr "to" target)* "}"
condition  := IDENT ("<" | ">" | "=") number
           | IDENT "is" "not" "empty"
           | IDENT "is" "empty"
           | IDENT "aligned" "with" IDENT
           | IDENT "has" IDENT
           | "true"
emit       := "emit" expr "to" target
emit_expr  := IDENT ("." IDENT)?
           | number
           | string
           | IDENT ("+" | "*" | "-") emit_expr
           | func "(" args ")"
emit_target := IDENT "." IDENT
              | "disk" string
              | "screen"
              | "world"
wire       := "wire" IDENT "." IDENT "to" IDENT "." IDENT

Keywords

Complete list — only 10 + 4 layer keywords:

KeywordPurpose
circuitDeclare an organ (V2)
bodyDeclare an organ (V3, same as circuit)
holdDeclare a neuron group
whenDeclare a route trigger
butGate a route with a condition
emitSend data out of a circuit
wireConnect circuits together
toDirection arrow (emit target, wire target)
streamDeclare a text stream (V3)
throughIterate through text characters (V3)
repeatStart a loop (V3)
untilEnd a loop (V3)
routeRoute data between holds (V3)

Layer Keywords

KeywordLayerLifetime
liveS1One tick — current perception
learnedL2One session — weights/skills
memoryE3Forever — episodic snapshots
awareC4Derived — convergence (who I am)

Shapes

ShapeMeaning
scalarOne neuron, one number
field [N]N neurons, a vector
field [N] [M]N×M neurons, a matrix (2D)
