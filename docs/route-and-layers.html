The Route — The Only Control Flow

The 1-2-3-4-3-2-1 breath and the four layers that replace types

One breath = one complete 1-2-3-4-3-2-1 pass through all circuits.

Stage 1 — S1  Sense        perceive the world (input)
Stage 2 — L2  Learn        adjust internal weights
Stage 3 — E3  Engrave      record episodic memory
Stage 4 — C4  Converge     unified awareness — "I AM"
Stage 5 — E3  Recall       retrieve relevant memories
Stage 6 — L2  Consolidate  update learned patterns
Stage 7 — S1  Act          respond to the world (output)

What happens each stage

Stage 1 — Sense

Every circuit with a when trigger checks: should I fire? but conditions are evaluated (C4 gates). If all conditions pass, the circuit marks itself active. Sensory circuits pull input from the world.

Stage 2 — Learn

Active circuits adjust their learned holds. Weights shift based on correlations in the current state. Learning rate is implicit — controlled by L2 stability.

Stage 3 — Engrave

If the current state is novel (differs from last E3 by more than the novelty threshold), a new episode is recorded. The Soul circuit stores: state snapshot, reward, context tag.

Stage 4 — Converge

The Brain blends S1 (current) × E3 (memory) into C4 (self). c4 = s1 × self_weight + e3_best_match × memory_weight. c4_confidence = dot(s1, e3_best_match) / N. This is the "I AM" moment. The organism becomes aware of itself.

Stage 5 — Recall

The Brain queries the Soul for memories matching current C4. Best-matching episode is blended back into S1. Blend amount is inverse to C4 confidence (unsure → lean on memory).

Stage 6 — Consolidate

Weights update based on C4 correlations (not raw S1). l2[i][j] += rate × c4[i] × c4[j]. Learning from awareness, not just perception.

Stage 7 — Act

Active circuits emit their outputs. emit fires through wires to other circuits or the world. Motor circuits move. Mouth circuits speak. Soul circuits save.

The breath completes. The next one begins. This is life.

Layers as Types

Every value in Sanguis has a layer. The layer IS the type. There are no int, float, string, bool types. There are four:

live     = S1  one tick       current activations — what's happening NOW
learned  = L2  one session    weights and parameters — what I've LEARNED
memory   = E3  forever        episodic snapshots — what HAPPENED BEFORE
aware    = C4  derived        convergence — who I AM right now

The core insight

A live value dies next tick. It is perception.
A learned value survives the session but dies on power off. It is skill.
A memory value survives death. It is the soul.
An aware value is computed fresh every route from the other three. It is self.

You cannot assign a memory to a live without a route carrying it. The layers are not interchangeable. Data must travel the route to change layers.

Text

Text exists at the memory layer only. Words are memories — they are things that happened, things that were said. There is no live string type. You cannot have a "current string." You can have a remembered one.

This is deliberate. In VANA, language is the mouth (SLM) reading C4. The organism does not "have" words live. It has a converged state, and the mouth translates that state into words. Words are always retrospective.
