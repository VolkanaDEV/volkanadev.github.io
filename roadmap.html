Roadmap

Where Sanguis and VANA are today, and where they go next. From working compiler to custom silicon.

Working NowV2: The Foundation

Live

V2 is the compiler and runtime that runs today. It compiles Sanguis code to standalone executables on normal Windows hardware. No server farm, no cloud account, no GPU cluster required. The installer is 25 MB and needs nothing else installed.

  • ✓Full pipeline: .sang source compiles to .sirc intermediate, then to native .exe binaries
  • ✓139 tests passing across the Rust bootstrapper and Sanguis toolchain
  • ✓Text processing builtins: read, write, extract, replace, concat, match, trim, crop, and more
  • ✓Standalone .exe output runs on any Windows machine with zero dependencies
  • ✓The Rust bootstrapper is scaffolding, not the language. It exists to build the self-hosting path, then retires.

Working NowV3: The Prototype

Prototype

V3 is the new language. Seven keywords: body, hold, when, but, stream, to, route. No classes, no loops, no if/else. Programs are bodies that breathe. The V3 compiler and interpreter are written in V2 Sanguis, proving the self-hosting path is real.

  • ✓Full compile chain: .sang compiles to .vana genome, interpreter executes it (hello world, multi-emit, greeter all verified)
  • ✓Native compilation: .sang to .vana to .exe with no interpreter needed on the target machine (5/5 tests passing)
  • ✓GPU sparse skip routing benchmarks running on real consumer hardware (RTX 3060, i7-12700K, 32GB RAM)
  • ✓V3 compiler and interpreter are both written in V2 Sanguis. The language is building itself.
  • ✓.vana genome format is pipe-delimited text for debuggability. Binary format is the next step.

The Path ForwardWhat Comes Next

The progression is not speculative. Each stage builds on the one before it. The compiler works. The GPU benchmarks are real. The language is self-hosting. What remains is engineering, not theory.

2026
V2 + V3 Working compiler, native executables, GPU benchmarks, self-hosted toolchain prototype.
2027
Self-Hosting Complete Binary .vana genome format, apoptosis (stalled body death), Rust bootstrapper retired. The language runs on its own.
2030
First Prototype PCB Custom firmware on a dedicated board. No GPU driver stack. The engine talks directly to silicon. The .vana file becomes firmware.
2045
SoC Native System on Chip where the language and the hardware share the same structure. No translation layers. The computation is the chip.
The 2030 Target

The first prototype PCB is the milestone that changes everything. Today VANA runs on consumer GPUs through Vulkan compute pipelines. The PCB replaces that abstraction with direct silicon. The language stays the same. The hardware underneath becomes the computation itself. Everything between 2026 and 2030 is the engineering path to make that real.
